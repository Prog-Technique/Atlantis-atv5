// Informações Gerais
// 1- Qual a localização do parque?
// O Thermas está localizado em Olímpia no interior do estado de São Paulo á 430 km da capital.

// 2- O Thermas dos Laranjais oferece desconto na compra de muitos ingressos? Quero organizar uma excursão. Teria alguma vantagem?
// O Thermas oferece condições especiais para grupos a partir de 15 pessoas. Para agendar uma excursão, entre em contato com uma das agências de turismo no site

// 3- Posso usar qualquer traje no parque?
// Para utilização das atrações são permitidos os seguintes trajes: Maiô, Biquíni, Sunga, Bermuda ou Shorts em tecidos apropriados (Lycra, Nylon, Tactel, Dry fit, não aderentes).
// Trajes na cor branca deverão ser forrados. Para crianças que necessitam do uso de fraldas convencionais, orientamos que utilize fraldas especificas para piscina;

// 4- Nas atrações radicais não é permitido utilizar:
// Relógios, óculos, pulseiras, anéis, piercing e acessórios em geral;
// Camisas ou camisetas;
// Bermudas ou trajes de banho com zíperes, botões ou adereços de metal;
// Demais trajes em tecido jeans ou similares.

// 5- Quais as formas de pagamento aceitas no Thermas dos Laranjais?
// Para compras de ingresso no site aceitamos somente o cartão de crédito. Na bilheteria do parque, nos bares e restaurantes aceitamos Thermas Card (cartão de Consumo), cartões de débito e crédito das bandeiras.

// 6- O Thermas dos Laranjais dá ingresso-retorno em caso de chuva ou mau-tempo?
// O Thermas opera normalmente com chuva desde que não haja risco para a segurança de nossos visitantes e não oferece devolução dos ingressos em caso de chuva e condições climáticas desfavoráveis.

// 7- Onde posso deixar meus objetos pessoais enquanto brinco no parque?
// O parque disponibiliza o aluguel de armários (pequenos e grandes) para que seus objetos sejam deixados em segurança, visto que o Thermas dos Laranjais não se responsabiliza por objetos deixados pelo parque. É necessário o pagamento do aluguel e de um depósito em dinheiro, que será ressarcido na devolução do item alugado. A locação de armários é um serviço opcional que o Thermas dos Laranjais oferece aos seus visitantes, que tem ainda a opção de deixar seus pertences no carro ou ônibus, descendo apenas com a roupa de banho. O acesso ao estacionamento é liberado. Ressaltamos que não nos responsabilizamos peles pertences deixados nos veículos e nas dependências do parque.

// 8- Posso alugar cadeira de rodas no Thermas dos Laranjais?
// Não. Somente disponibilizamos cadeiras de rodas para mobilidade interna no parque na chegada e saída.

// 9- Eu vou ao parque apenas para acompanhar meus filhos, não irei a nenhuma atração. Terei desconto no valor do ingresso?
// Não. Infelizmente, não temos como controlar se nossos visitantes vão ou não frequentar as atrações. E, você utilizará as dependências do parque, como vestiários e banheiros. Por isso, o valor do ingresso é o mesmo, sem descontos.

// 10- Qual altura permitida nas atrações?
// Nas atrações radicais a altura mínima permitida a partir de 1,20m.

// 11- Qual o procedimento em caso de objetos / documentos perdidos no Thermas dos Laranjais?
// Você deverá ir até a sala do achados e perdidos, na entrada do parque, e registrar essa perda e as características do objeto ou documento perdido. Se notar a perda em casa, é só entrar em contato com nosso Atendimento pelo telefone 17 3279-3500 ramal 565. Ressaltamos que o Thermas dos Laranjais não se responsabiliza por esses objetos.

// 12- Há alternativas de alimentação dentro do parque?
// Existem mais de 30 estabelecimentos de alimentação com tipos variados de alimentos.

// 13- Posso entrar com alimentos e bebidas no Thermas dos Laranjais?
// Visando a segurança de nossos visitantes a entrada de alimentos e bebidas no parque é restrita e controlada, não sendo permitida a entrada no interior do parque com caixas de isopor e coolers.

// Essa medida se justifica por estarmos localizados em uma região de clima quente, onde são necessários métodos especiais para conservação dos alimentos, para evitar intoxicações e mal-estar. Os produtos comercializados nos quiosques do parque são armazenados corretamente de forma a garantir a qualidade do que é consumido pelos nossos visitantes. O Thermas não se responsabiliza por quaisquer problemas que possam ocorrer com alimentos vindos de fora e que não passem pela nossa inspeção. Lembramos que a entrada com mamadeiras, papinhas infantis e alimentos comprovado por orientação médica é permitida. E o acesso ao estacionamento e estabelecimentos vizinhos é liberado.

// 14- O Thermas dos Laranjais tem posto médico?
// Sim. O Thermas possui um posto médico com médico e enfermeiro sempre de plantão para prestar atendimento aos visitantes que necessitem de cuidados em caso de mal-estar, alteração de pressão, hipoglicemia, desidratação e insolação. Os pequenos traumas como quedas, escoriações, ferimentos, entorses e luxações também são atendidos prontamente por nossa equipe.

// 15- É preciso fazer algum exame médico antes de entrar no parque?
// Não. Por possuir um moderno sistema de tratamento e purificação de água, por ozônio, que dispensa a necessidade de exames médicos.

// 16- Qual o valor do estacionamento?
// Estacionamento gratuito.

// 17- Qual o procedimento de crianças perdidas no parque?
// Nossos atendentes são treinados para em caso de crianças perdidas direcioná-las até O CAC- Centro de Atendimento a Criança, próximo a portaria 2 (relógio) onde essas crianças ficam sob os cuidados dos atendentes até o encontro do responsável.
